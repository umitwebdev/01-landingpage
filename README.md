# 01-landingpage

Interaktive Landingpage mit Klick-Counter und Schwellwert-Anzeige.

**Live:** [umitwebdev.github.io/01-landingpage](https://umitwebdev.github.io/01-landingpage/)

## Features

- Live-Update der Anzeige bei jedem Klick (DOM-Manipulation ohne Page Reload)
- Drei Schwellwert-Stufen: 0–9 normal, 10–19 🔥, 20+ 🚀
- Reset-Funktion
- Vollständig Vanilla JavaScript, keine Frameworks

## Tech-Stack

- HTML
- CSS
- Vanilla JavaScript
- DOM API

## Konzepte

- DOM-Manipulation über `querySelector` und `textContent`
- Control Flow mit `if / else if / else`
- State-Management im Module Scope
- Refactoring nach DRY und Single Responsibility

## Setup

```bash
git clone https://github.com/umitwebdev/01-landingpage.git
cd 01-landingpage
```

Dann `index.html` im Browser öffnen oder mit Live Server starten.
